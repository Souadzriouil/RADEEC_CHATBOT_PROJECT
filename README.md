# 🤖 RADEEC RAG Chatbot

## 📌 Overview
This project is a RAG-based AI chatbot designed to automate customer service using semantic search and Large Language Models (LLMs).

## 🎯 Objective
The chatbot helps answer customer queries related to billing, consumption, and frequently asked questions by combining information retrieval and AI-generated responses.

## 🚀 Features
- Retrieval-Augmented Generation (RAG)
- Semantic search for relevant information
- LLM-based response generation
- Customer support automation
- Interactive chatbot system

## 🛠️ Technologies
- Python
- LangChain
- LLaMA 3
- Sentence Transformers
- Streamlit
- SQLite

## 📂 Project Structure
- `chatbot.py` → main chatbot logic  
- `database.py` → database management  
- `data_data.txt` → knowledge base
